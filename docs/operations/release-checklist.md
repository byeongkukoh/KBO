# Release Checklist

## Documentation

- MVP 범위 문서와 실제 구현 범위가 일치하는가?
- KBO 공식 원천 데이터 정책이 운영 문서에 반영되어 있는가?
- 고급 지표 제공 범위가 의존성 문서와 충돌하지 않는가?

## Backend Readiness

- API가 선수, 팀, 경기 조회의 핵심 응답을 제공하는가?
- 스크래핑 또는 배치 작업이 마지막 성공 시점을 기록하는가?
- 원천 데이터 식별자와 내부 식별자 매핑이 검증되었는가?

## Frontend Readiness

- 선수, 팀, 경기 화면의 핵심 사용자 흐름이 동작하는가?
- 데이터 최신성 또는 마지막 갱신 시점이 노출되는가?
- 경기 상세 화면이 합의된 raw 수준까지 정보를 보여주는가?

## Data Readiness

- MVP에 필요한 KBO 페이지 유형이 실제로 수집 가능한가?
- 시즌별 선수/팀 기본 기록이 누락 없이 적재되는가?
- Tier 2 이상 고급 지표는 필요한 리그 평균과 보정 데이터가 확보된 경우에만 노출되는가?

## Go / No-Go Gates

- 선수 이적 이력, 리그 평균 데이터, 경기 이벤트 로그 수준이 MVP 가정과 다르면 범위를 재조정했는가?
- 문서에 남아 있는 open question 중 출시를 막는 항목이 명확히 처리되었는가?
