# Service Overview

## Purpose

KBO 선수 기록과 팀 기록을 시즌별, 경기별로 쉽게 확인할 수 있는 웹 서비스를 구축한다. 사용자는 여러 시즌의 누적 기록과 개별 경기 기록을 한곳에서 탐색할 수 있어야 하며, 시즌 진행 중에는 매일 경기 결과가 반영된 최신 데이터를 확인할 수 있어야 한다.

이 서비스의 1차 목표는 기록 조회 경험을 빠르고 일관되게 제공하는 것이다. 이후에는 축적된 기록 데이터를 기반으로 팀 순위 예측이나 경기 결과 예측 같은 분석 기능으로 확장할 수 있는 기반을 마련한다.

## Target Users

- KBO 팬: 특정 선수나 팀의 시즌 성적과 경기별 기록 흐름을 쉽게 확인하고 싶어하는 사용자
- 기록 중심 사용자: 시즌별 누적 기록, 경기별 상세 기록, 팀 단위 비교 데이터를 함께 보고 싶은 사용자
- 야구 콘텐츠 작성자 및 분석형 사용자: 일일 업데이트되는 데이터를 바탕으로 기사, 콘텐츠, 분석 노트를 작성하려는 사용자

## Core Value Proposition

이 서비스는 KBO 기록 데이터를 시즌 단위와 경기 단위로 구조화해 보여주고, 시즌이 진행되는 동안에는 하루 1회 자동 업데이트를 통해 최신 상태를 유지한다. 단순한 기록 보관소가 아니라, 사용자가 특정 시즌, 팀, 선수, 경기를 기준으로 원하는 기록을 빠르게 찾고 흐름을 이해할 수 있는 탐색형 기록 플랫폼을 지향한다.

## MVP Scope

- 시즌별 선수 기록 조회
- 선수 상세 화면에서 데뷔 시즌부터 현재까지의 시즌별 기록 조회
- 선수 기본 정보 제공(현재 소속 팀, 포지션, 생년월일 또는 나이, 프로필 정보)
- 선수 이력 정보 제공(팀 이동 및 이적 이력)
- 타자와 투수 모두 기본 기록과 고급 통계 지표 제공
- 시즌별 팀 기록 조회
- 경기별 선수 및 팀 기록 조회
- 시즌, 팀, 선수, 경기 기준의 기본 탐색 및 필터링
- 시즌 진행 중 하루 1회 데이터 스크래핑 및 데이터베이스 반영
- 웹 프론트엔드와 API 백엔드 기반의 기록 조회 서비스 제공

## Out of Scope

- 팀 순위 예측, 경기 결과 예측, 선수 성과 예측 기능
- 고급 개인화 기능(즐겨찾기, 알림, 추천)
- 관리자 전용 운영 화면
- 실시간 경기 중계 수준의 즉시 반영 기능
- 커뮤니티, 댓글, 소셜 공유 기능
